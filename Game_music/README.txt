DEAD WORLD - Game Audio Assets
===============================

FOLDER STRUCTURE
----------------

Ambient/
  Background music and atmospheric sounds used in the main menu and gameplay.
  NOTE: "julius_galla__atmosphere-horror-2-loop.wav" is directly referenced in code - DO NOT rename.
  Other files: ambient_horror_loop_01.wav, ambient_retro_horror_loop.wav,
               ambient_horror_jumpscare.mp3, ambient_heartbeat.mp3
  Sources: Freesound.org (CC0 / free licenses)

Gun Sounds/
  Single gunshot and reload sounds for shooting mechanics.
  Files: gun_pistol_shot_01.wav, gun_pistol_shot_02.wav, gun_rifle_shot_01.wav,
         gun_pistol_mix_01.mp3, gun_ak47_fire_01.mp3,
         gun_reload_01.mp3, gun_reload_02.mp3
  Sources: Freesound.org (CC0 / free licenses)

Punch_Sounds/
  Melee impact sounds for close combat hits.
  Files: punch_heavy_01-05.ogg (solid punch impacts)
         punch_flesh_01-03.ogg (soft body/flesh impacts)
  Source: Kenney.nl Impact Sounds Pack (CC0 - Public Domain)
          https://kenney.nl/assets/impact-sounds

Zombie Sounds/
  Zombie groans, screams, and attack sounds played when zombies appear.
  Files: zombie_moan_01.mp3, zombie_scream_female_01.mp3, zombie_screech_01.mp3,
         zombie_sfx_01.mp3, zombie_grunt_01.mp3, zombie_attack_01.mp3
  Sources: DRAGON-STUDIO via Pixabay (royalty-free),
           Zombioo open-source project (MIT)

Blood sound/ (unused in code - archived for future use)
  Various blood/gore sound effects.

Roar Sounds/ (unused in code - archived for future use)
  Monster roar sound effects.
