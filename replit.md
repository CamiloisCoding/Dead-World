# Dead World

## Overview
Dead World is a post-apocalyptic text adventure game built with Python and Pygame. Players navigate through a zombie-infested world, explore rooms, collect items, and engage in combat. The game features a terminal-style UI with CRT effects.

## Project Architecture
- **Language**: Python 3.12
- **Framework**: Pygame 2.6.1
- **Type**: Desktop GUI application (VNC output)
- **Entry Point**: `dead_world_intro_v_omega.py` (single-file game)

## How to Run
The game runs via the "Dead World Game" workflow using VNC display output.

## Key Features
- Text adventure with room navigation (German language)
- Combat system with weapons, enemies, and QTE mechanics
- Inventory management
- CRT/terminal visual effects
- Resolution scaling support

## System Dependencies
- SDL2, SDL2_image, SDL2_mixer, SDL2_ttf
- X11 libraries (libX11, libXext, libXrender, etc.)
- fontconfig, freetype, libGL
